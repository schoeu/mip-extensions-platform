# mip-cs-meiqia

mip-cs-meiqia 组件说明：美洽在线客服组件——美洽是一个流行的网页在线客户服务`Customer Service`系统，旨于为企业提升PC及移动端访问用户转化率。

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/extensions/platform/v1/mip-cs-meiqia/mip-cs-meiqia.js

## 示例

### 基本用法
```html
<mip-cs-meiqia eid="your-meiqia-eid"></mip-cs-meiqia>
```

## eid

### 美洽官方分配的用户标识

说明：美洽后台分配给您的entid
必选项：是
类型：字符串
取值范围：任何
默认值：无
