## 属性

### ansArea
说明：回复区域
必填：是
格式：字符

### ansAreaHeight
说明：回复区域高度
必填：是
格式：数字

### hiddparam
说明：隐藏的参数配置
必填：是
格式：字符串

### showparam
说明：显示的参数配置
必填：是
格式：json串
