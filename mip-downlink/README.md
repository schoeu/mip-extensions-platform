# mip-downlink

mip-downlink 下载软件

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-downlink/mip-downlink.js

## 示例

### 基本用法
```html
<mip-downlink iurl="http://www.baidu.com/getdownlink" tag="123">
    <span class="adownlink">下载</span>
</mip-downlink>
```

## 属性

### iurl

说明：接口地址
必选项：是
类型：string
默认值：无
