# mip-shouji-down

mip-shouji-down 下载站定制组件，主要用于导航切换与下载按钮切换。  

标题|内容
----|----
类型|通用
支持布局|N/S
所需脚本|https://c.mipcdn.com/extensions/platform/v1/mip-shouji-down/mip-shouji-down.js

## 示例

### 基本用法
```html
<mip-shouji-down down="URL"></mip-shouji-down>
```

## 属性

### down

说明：网址  
必选项：否
